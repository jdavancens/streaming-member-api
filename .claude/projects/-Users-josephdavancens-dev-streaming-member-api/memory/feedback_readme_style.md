Don't add "X is **not** required" sentences to README prerequisite sections. List only what the reader needs; let the absence speak for itself.

**Why:** User called this out as unnecessary clutter. A prerequisites list should be affirmative.

**How to apply:** When updating prerequisites after removing a dependency, just delete the old line — don't add an explanatory "no longer needed" sentence.
